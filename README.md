# What is SL Survey

SL Survey is Simple Survey Application

This Application is composed of [Rails](http://rubyonrails.org/)(user-pages,admin-pages) + UI [Boostrap](http://getbootstrap.com)

## Server Requirements

[Rails](http://rubyonrails.org/) version 6
